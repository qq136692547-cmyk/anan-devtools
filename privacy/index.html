Privacy, the short version.

The short version

We do not collect any personal data. We do not run any analytics. We do not set any tracking cookies. We do not log your input. We do not sell anything to anyone.

How the site works

anan.zh.kg is a static website hosted on GitHub Pages. Every page is a plain HTML file. Every tool runs in your browser using JavaScript. There is no server-side processing of your input.

When you paste text into a tool, decode a JWT, or generate a UUID, the operation happens locally on your computer. The data never leaves your machine.

What GitHub Pages sees

Like every website, GitHub Pages logs basic request metadata (IP address, user agent, timestamp) for security and operational purposes. This is standard web server logging and is not under our control. See GitHub Pages documentation for details.

Third-party services

We load two external resources:

  • Google Fonts (IBM Plex Sans, IBM Plex Mono) — standard web fonts. Google may log requests to fonts.googleapis.com. We could self-host, but the bundle size benefit is not worth the complexity.
  • That is it. No analytics, no ads, no trackers.

Cookies

We do not set any cookies. Period.

Ads

There are no ads on this site. We may add Google AdSense in the future, but if we do, we will update this page first and make it obvious.

Changes to this policy

If we change this policy, we will update the "last updated" date below. The current version was last updated on 24 June 2026.

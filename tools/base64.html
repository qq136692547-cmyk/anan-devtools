What is Base64?

Base64 is a binary-to-text encoding scheme that represents binary data using a set of 64 ASCII characters: A-Z, a-z, 0-9, +, /, and = for padding.

Common uses

  • Embedding images in HTML/CSS
  • Encoding email attachments (MIME)
  • JWT tokens

TOOL 13

Cron Expression Parser

Parse cron expressions and see the next 5 run times. Human-readable explanation of each field.

Examples: 0 9 * * 1-5 (weekdays at 9am) · 0 0 1 * * (1st of month) · */5 * * * * (every 5 min)

—

    Cron syntax

    * * * * *
| | | | |
| | | | +- weekday (0-6, Sun=0)
| | | +--- month (1-12)
| | +----- day (1-31)
| +------- hour (0-23)
+--------- minute (0-59)

    Special characters

    • * — any value
    • , — list (1,3,5)
    • - — range (1-5)
    • / — step (*/15 = every 15)

TOOL 06

HTML Escape / Unescape

Escape and unescape HTML entities. Safe for templates, JSX and string literals.

What gets escaped

  • & → &amp;
  • < → &lt;
  • > → &gt;
  • " → &quot;
  • ' → &#39;
  • / → &#x2F;
  • ` → &#x60;
  • = → &#x3D;
